A little history...

1989: HTML introduced by Tim Berners Lee

SGML + hyperlinks

1992 HTML+ by Dave Ragget

Arena browser

forms, CHSS, background images, tables

1995 JavaScript & DOM Level 0 from Netscape

Very limited, only a few elements scriptable

1995-1997: Scott Issacs @ Microsoft

CSS 2-D positioning, ActiveX, DHTML, XML Data Islands, iframe

The Dark Age 1997-2004

1997: HTML Components from Microsoft

1999: Microsoft wins browser wars?

IXmlHttpRequest

A little progress on CSS implementation

ActionScript

2000: Mozilla XHR in Gecko (pre-firefox)

2004: Gmail (Paul Buchheit)

Birth of modern AJAX

2005: Dojo & prototype.js

2006: jQuery, MooTools, & YUI

Abstracts away the hard parts of DOM & AJAX.

2009+: Javascript Explodes

node.js, Angular 1, underscore

Knockout, Backbone, npm

Ember, React, Polymer

2016: Vue 2, riot.js, Angular 2
